# Amazon Homepage Clone

A static Amazon homepage clone built using HTML and CSS.
This project was created to practice webpage layout, navigation bars, product cards, image sections, hover effects, footer structure, and basic responsive design.

## Live Demo

https://ecommerce-homepage-clone.vercel.app/

## Features

* Amazon-style navigation bar
* Search bar UI
* Location, language, account, orders, and cart sections
* Hero/banner section
* Product category cards
* Multiple shopping sections
* Footer layout
* Hover effects
* Basic responsive layout using media queries

## Tech Stack

* HTML
* CSS
* Font Awesome
* Vercel for deployment

## What I Learned

* How to structure a large static webpage using HTML
* How to create a navbar similar to a real ecommerce website
* How to use Flexbox for layout
* How to create product/category card sections
* How to use background images and external images
* How to improve layout using `flex-wrap`, `min-width`, and media queries
* How to deploy a static website using Vercel
